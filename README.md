# Dockerized Web App

## Project Overview
This project demonstrates how to build and containerize a simple Node.js web application using Docker.  
It includes environment variables, a health check endpoint, and a multi-stage Dockerfile to make it production-ready.

### What I Built
- A simple **Node.js Express app**
- Containerized using **Docker**
- Added:
  - Multi-stage Dockerfile
  - Environment variables via `.env.`
  - Health check endpoint (`/health`)

### Skills Demonstrated
- Containerization
- App packaging
- Basic deployment readiness
